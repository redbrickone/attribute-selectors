CSS Selectors

The good, the bad and the CSS...

Basic CSS Selectors

*

Asterisks apply to every element on the page


          * {
            background-color: red;
            margin: 0;
          }

#id

Hash symbol targets elements by id


          #element-id {
            position: relative;
            margin: 0;
          }

.class

Dot symbol targets elements by class name


          .element-class {
            position: relative;
            margin: 0;
          }

X Y

Descendant Selector

This allows for more specificity in selectors


          .element-class a {
            color: #FFFFFF;
            font-size: 2em;
          }

X + Z

adjacent selector.

It will select only the element that is immediately preceded by the former element.

only the first element with class my-element after each ul will have white text and size 2em


          ul + .my-element {
            color: #FFFFFF;
            font-size: 2em;
          }

A > B

This will only select direct children


          div.my-class#container > ul {
            border: 1px solid black;
          }

          <div>
            <ul>
              this one IS affected
              <li>
                <ul>
                 this one not affected
                </ul>
              </li>
            </ul>
          </div>

W ~ X

Less strict version of the X + Y selector

Any p elements, as long as they follow a ul.


          ul ~ p {
            padding: 5px;
            font-size: 2em;
          }

          <ul>
            <p>
              this one is affected
              <li>
                <p>
                 this one is also affected
                </p>
              </li>
            </p>
          </ul>

Attribute Selectors

Here's where things get fun

CSS [attribute] Selector

How to select an element by their href, download attr, etc...


          a[download] {
            background-color: blue;
          }

The [attribute=”value”] Selector

This selects an html element with a specific attribute AND a specific value. 


                  a[target="_blank"] {
                      color: blue;
                      font-weight: bold;
                  }

The [attribute~=”value”] Selector

This selector is similar to the previous selector but it selects a specific attribute and value but the value must be a space-separated word.


          [title~="namespace"] {
            color: white;
              font-weight: 200;
          }

The [attribute|=”value”] Selector

This selector selects an element with a specific attribute but the value begins with a specific word. It must be a whole word or be separated from other by a hyphen.


          a[class|="top"] {
              background: #CCCCCC;
          }

The [attribute^=”value”] Selector

This is just like the last selector, however the value does not have to be a whole word.


          h2[class^="topbar"] {
              background: red;
          }

The [attribute$=”value”] Selector

This selector selects an element with a specific attribute that ends with a specific value, rather than beginning with that value.


              a[class$="st"] {
                  background: green;
              }

will select an element with a class of "greatest".

The [attribute*=”value”] Selector

This selector selects any html element that has a specific attribute and contains a string of textanywhere in the value, so it does not need to begin or end with it as long as it contains the value.


              a[class*="te"] {
                  background: blue;
              }
